# Browser_data_erasure_tool
浏览器数据清除脚本，可以检查浏览器中有没有存储特定网址的密码，然后实行数据的清除，主要是用于攻防演练中蓝队防止钓鱼获取浏览器密码。
# 查找浏览器是否存储特定网址的密码
在websites.txt中填入要查找的特定网址，一般是单位内重要系统比如OA等。
